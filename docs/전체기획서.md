1. 개발목적 : 현재 운영중인 유료사이트의 자료의 일부자료를 불특정 다수에게 무료 공개하며, 사용자의 반응을 피드백 받음
- 사용자가 많은 교재 및 단원 수 집계
- 추가로 제작을 요청하는 교재들
- 시간대별 접속인원
- 추가적인 요청 받기

2. 페이지 레이아웃
- 메인페이지
  3단 레이아웃(좌측:교재목록, 검색, 요청/ 중앙:pdf파일 미리보기/ 우측: 교재요청 및 사용 순위, 공지사항)
  좌측 : 최상위폴더(로컬)>교재명>단원명>그룹명>유형명.pdf  - pdf로만 자료공개함 / 검색해서 해당교재만 보이기, 검색해서 교재가 없으면 해당 교재 요청하기 입력>요청이 많은 교재는/ 우측패널에 표시(사용자추가요청클릭)  > 관련 내용은 관리자페이지에서 관리리
  중앙 : pdf자료 미리보기 화면
- 관리자 페이지
  좌측 교재 목록(클릭수많은순서), 우측 요청교재(요청수많은순서)의 관리
  공지사항 관리
  기타, 사용자 성향 및 동태 파악 및 관리

3. 1차 개발 목표
    - 기본셋팅
    - 로컬과 드롭박스 연동 (좌측패널 우선 구현)
