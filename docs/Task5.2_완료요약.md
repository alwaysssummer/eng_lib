## ✅ 구현된 기능

### 1️⃣ 교재 목록 페이지 (`/admin/textbooks`)

- 📊 **통계 카드 3개**
  - 전체 교재 수
  - 전체 클릭수
  - 전체 파일 수

- 🔍 **검색 기능**
  - 교재명으로 실시간 검색
  - 입력 즉시 반영

- 🔄 **정렬 기능**
  - 교재명 (가나다순)
  - 클릭수 (많은순/적은순)
  - 헤더 클릭으로 토글

- 📋 **교재 목록 테이블**
  - 교재명
  - 클릭수 Badge
  - 파일 수
  - 활성화 상태
  - 생성일
  - 작업 버튼 (활성화/비활성화, 상세보기)

### 2️⃣ 교재 상세 페이지 (`/admin/textbooks/[id]`)

- 📈 **통계 카드 3개**
  - 총 클릭수
  - 전체 파일 수
  - 평균 클릭수

- 📊 **일별 클릭 추이 그래프**
  - 최근 30일 데이터
  - recharts BarChart
  - 반응형 디자인
  - 툴팁으로 상세 정보

- 📁 **파일 목록 테이블**
  - TOP 3 메달 표시 (🥇🥈🥉)
  - 클릭수 많은 순 정렬
  - 파일명, 경로, 클릭수, 상태
  - 최종 수정일

### 3️⃣ API 엔드포인트

- **GET** `/api/admin/textbooks` - 교재 목록 (정렬/검색)
- **PUT** `/api/admin/textbooks` - 활성화 토글
- **GET** `/api/admin/textbooks/[id]` - 교재 상세 정보
